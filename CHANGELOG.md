# Changelog v2.2.0

## Feature

- add the support for the customised properties of general databases
- 增加对普通数据库自定义属性的支持

- support
  - [x] `title`, which is the first column of the database
  - [x] 'Text'
  - [x] 'Number'
  - [x] 'Date'
  - [x] 'Checkbox'
  - [x] 'Select'
  - [x] 'Multi-select'
  - [x] 'URL'
  - [x] 'Email'
  - [x] 'Phone'
  - [x] 'File' (**Only support external embedded files**)
